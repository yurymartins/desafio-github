Conceitos do Git:

SHA1 - SHA: Secure Hash Algorithm (Algoritmo de Hash Seguro)
Gera conjunto de caracteres identificador de 40 dígitos.

Blob (binary large object): contém o conteúdo do arquivo (sha)
Tree: Um objeto da árvore do Git (tree) cria a hierarquia entre arquivos em um repositório do Git. Você pode usar o objeto da árvore do Git para criar a relação entre diretórios e os arquivos que eles contêm.
Commit: é o objeto que vai juntar tudo, dando sentido a alteração efetuada.
